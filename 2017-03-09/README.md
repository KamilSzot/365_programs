Misc
====

Making Gemfile:

    gem install gemrat
    gemrat curses

plus line:

    source 'https://rubygems.org'

added manually.


Installing deps form Gemfile:

    gem install bundle
    bundle install

        
Doesn't work with git bash (MinGW, mintty) because of "Redirection is not supported." error (curses?).

Works with cmd and powershell.
